# Galactic Defender

## How To Play The Game

Shoot all the enemy ships down to progress and accumulate as much points as possible.

### Install Depedencies

```bash
pip install -r requirements.txt
```

### Start Game

```bash
python main.py
```

### Controls

- Press `Q` to Quit
- Press or Hold `spacebar` to shoot
- &larr; &uarr; &rarr; &darr; to Move
